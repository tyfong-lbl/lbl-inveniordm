This page provides answers to frequently asked questions about {{ config.THEME_SITENAME }}.

What is {{ config.THEME_SITENAME }}?

{{ config.THEME_SITENAME }} is a data repository service for Berkeley Lab users. It allows researchers to publish, preserve, and share their research data.

How do I upload data?

To upload data, you need to:

  1. Log in to your account
  2. Click on "Upload" in the main navigation
  3. Fill in the required metadata
  4. Upload your files
  5. Submit your deposit for review

What file formats are supported?

{{ config.THEME_SITENAME }} supports all file formats. However, we recommend using open and well-documented formats to ensure long-term accessibility of your data.

What is the maximum file size?

The maximum file size for a single file is 50GB. For larger datasets, please contact our support team at {{ config.SUPPORT_EMAIL }}.

How do I cite data from {{ config.THEME_SITENAME }}?

Each dataset in {{ config.THEME_SITENAME }} has a unique DOI that can be used for citation. The citation information is provided on the record page of each dataset.
